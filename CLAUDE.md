# Always With

When starting a session on this project, read `BACKLOG.md` for the current list of improvement ideas, suspected bugs and pending experiments. When the user says something like "lavoriamo su qualcosa dal backlog", propose the next candidates from that file.

When an item from `BACKLOG.md` is implemented and shipped, move it to the **Done** section with the version that contains it.

Local-only reference material (design screenshots, sketches, notes that the user pastes during sessions) lives in the gitignored `references/` directory. When a backlog item refers to "see screenshots in `references/`", look there for the relevant files.
